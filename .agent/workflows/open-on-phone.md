---
description: Открыть локальный dev-сервер для доступа с телефона в одной сети
---

Этот воркфлоу помогает запустить проект так, чтобы он был доступен по локальной сети, и определить нужный URL.

1. **Определить IP-адрес ноутбука**
Используй команду `ipconfig` и найди IPv4-адрес в активном сетевом адаптере (обычно Wi-Fi).

2. **Определить команду запуска**
Проверь `package.json` на наличие скрипта `dev`.

3. **Запустить сервер на 0.0.0.0**
// turbo
Запусти команду `npm run dev -- -H 0.0.0.0` (для Next.js) или аналогичную, которая биндит сервер на `0.0.0.0`.

4. **Проверить порты**
Убедись, что порт (обычно 3000) прослушивается на всех интерфейсах с помощью `netstat -ano | findstr :3000`.

5. **Предоставить Network URL**
Сформируй URL в формате `http://<IP_ноутбука>:<порт>` и покажи его пользователю.
